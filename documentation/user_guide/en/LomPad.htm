LomPad system allows the referencing of learning resources using metadata according to the international IEEE LOM standard and also to SCORM, CANCORE and Normetic application profiles.

1. LomPad Interface:

PALOMA interface is composed of a menu bar and section:

  • The Menu Bar gives access to a series of drop-down menus containing LomPad's commands.
    • File : The File menu contains the following functions : New, Open, Save, Save As, View XML and Quit.
    • Language : Choose between French or English for the Interface.
    • Profiles : Choose between IEEE LOM, CanCore, SCORM or Normetic (required, recommended or all elements).
    • Help : Access the IEEE Reference document.
  • The section is :
    • Metadata section: Allows to consult or edit metadata..

2. Manage LOM:

You may work on LomPad to consult, add, modify or delete LOM records and their elements.

2.1 Create a LOM:

  • To create a new LOM, click on New command on the File menu.

2.2 Complete Metadata:

Data elements in the LOM Profile (IEEE LOM, CanCore, SCORM and Normetic) are grouped into different categories. Note that the different profiles do not include all categories. See references at the end of this document for more details.

The 9 (nine) categories are:

  1. The General category groups the general information that describes the learning object as a whole.
  2. The Life Cycle category groups the features related to the history and current state of this learning object and those
    who have affected this learning object during its evolution.
  3. The Meta-Metadata category groups information about the metadata instance itself (rather than the learning
    object that the metadata instance describes).
  4. The Technical category groups the technical requirements and technical characteristics of the learning object.
  5. The Educational category groups the educational and pedagogic characteristics of the learning object.
  6. The Rights category groups the intellectual property rights and conditions of use for the learning object.
  7. The Relation category groups features that define the relationship between the learning object and other related
    learning objects.
  8. The Annotation category provides comments on the educational use of the learning object and provides
    information on when and by whom the comments were created.
  9. The Classification category describes this learning object in relation to a particular classification system.

To complete metadata (LOM):

  • Complete the metadata you have on this learning object by filling the corresponding fields. For element (category or element) having multiple cardinality, see sections below.
  • Once completed, don't forget to click on Save at the bottom of the Metadata section.

2.2.1. Add an element or a sub-element:

For some metadata elements, you may have multiple cardinality:

  • Click on the plus (+) button on the right of the field corresponding to the metadata (or its leafs) to add a new element (or sub-element).

2.2.2. Remove an element of sub-element:

  • Click on the minus (-) button on the right of the metadata (or its leafs) to remove an element (or sub-element).

2.2.3. Add a metadata category :

As well, for some categories Educational, Relation, Annotation or Classification, you may have multiple cardinality:

  • Select the category in the Metadata section: Educational, Relation, Annotation or Classification.
  • Click on the plus (+) button on the left part of the Metadata section.

2.2.4. Remove a category:

  • Select the category in the Metadata section: Educational, Relation, Annotation or Classification.
  • Select the category on the left part of the Metadata section you want to remove.
  • Click on the minus (-) button on the left part of the Metadata section.

2.3. Open a LOM:

  • Click on Open command on the File menu.
  • This command opens dialog box where you can locate and open the document using its file name.

2.4. Save a LOM:

To store your LOM in a folder (directory) under an appropriate name, click on one of the save commands ( Save or Save As ) following Windows® standards. This system save the metadata in XML format.

2.5. View the LOM in XML format:

  • Click on XML command on the File menu.
  • The XML document will be displayed with color-coded root and child elements. Clicking on the plus (+) or minus (-) signs on the left will expand or collapse the metadata element structure.

2.6. Quit a LomPad:

  • Click on Quit command on the File menu.
