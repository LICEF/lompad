LomPad

    

The goal of this help is to facilitate the use of features in the LomPad. This system allows the referencing of the resources with metadata according to the international IEEE LOM, SCORM, CANCORE and Normetic standards.

A) Menu Bar

  • The Menu Bar gives access to a series of drop-down menus containing LomPad's commands.
    • File : The File menu contains the following functions : New, Open, Save, Save As, View XML and Quit.
    • Language : Choose between French or English for the Interface.
    • Profiles : Choose between IEEE LOM, CanCore, SCORM or Normetic (required, recommended or all elements).
    • Help : Access the IEEE Reference document.

B) Create, open and save LOM

1 To Create a LOM :

  • To create a new LOM, click on New command on the File menu.

1.1 To enter Metadata for a new LOM:

  • Complete the metadata according to the LOM Profile you have previously selected.
  • Once completed, don't forget to click on "Save" at the bottom of the metadata frame.

LOM Profile metadata

Data elements in the LOM Profile (IEEE LOM, CanCore, SCORM and Normetic) are grouped into different categories. Note that the different profiles do not include all categories.

The 9 (nine) categories are:

  1. The General category groups the general information that describes the learning object as a whole.
  2. The Life Cycle category groups the features related to the history and current state of this learning object and those
    who have affected this learning object during its evolution.
  3. The Meta-Metadata category groups information about the metadata instance itself (rather than the learning
    object that the metadata instance describes).
  4. The Technical category groups the technical requirements and technical characteristics of the learning object.
  5. The Educational category groups the educational and pedagogic characteristics of the learning object.
  6. The Rights category groups the intellectual property rights and conditions of use for the learning object.
  7. The Relation category groups features that define the relationship between the learning object and other related
    learning objects.
  8. The Annotation category provides comments on the educational use of the learning object and provides
    information on when and by whom the comments were created.
  9. The Classification category describes this learning object in relation to a particular classification system.

1.1.1 To add an element or a sub-element:

For some metadata elements, you may have many instances of an element or sub-element:

  • Click on the plus (+) button on the right of the metadata (or its leafs) to add a new element (or sub-element).

1.1.2 To delete an element of sub-element:

  • Click on the minus (-) button on the right of the metadata (or its leafs) to delete an element (or sub-element).

1.1.3 To add a metadata category (or duplicate a category):

As well, for some categories Educational, Relation, Annotation or Classification , you may have many instances:

  • Select the tag corresponding to the category in the Metadata section(entitled "Profile").
  • Click on the plus (+) button on the left part of the Metadata section.

1.1.4 To delete a duplicate category:

  • Select the tag corresponding to the category in the Metadata ("Profile") section Educational, Relation, Annotation or Classification .
  • Select a duplicated category on the left part of the Metadata section.
  • Click on the minus (-) button on the left part of the Metadata section.

2 To Open a LOM:

  • Click on Open command on the File menu.
  • This command opens a Windows® Open dialog box where you can locate and open the document using its file name.

3 To Save a LOM:

To store your LOM in a folder (directory) under an appropriate name, click on one of the save commands ( Save or Save As ) following Windows® standards.

4 To view the LOM in XML format:

  • Click on XML command on the File menu.
  • The XML document will be displayed with color-coded root and child elements. Clicking on the plus (+) or minus (-) signs on the left will expand or collapse the metadata element structure.

5 To Quit the LorPad:

  • Click on Quit command on the File menu.
