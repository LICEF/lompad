LomPad permet de référencer les objets d'apprentissage selon les métadonnées du LOM (Learning Object Metadata) à l'aide des normes internationales d'IEEE, LOM, de SCORM, de CANCORE et de Normetic. De plus, ce système sauve les métadonnées en format XML.

1. Interface de LomPad

L'interface de LomPad est composé d'une barre de menu et d'une section métadonnées.

  • La barre de menus donne accès à une série de menus composés des principales commandes du LomPad.
    • Fichier : Le menu fichier donne accès aux commandes suivantes :Nouveau, Ouvrir, Sauver, Enregistrer sous , Voir XML et Quitter
    • Langue : Il est possible de choisir le type de langue : Anglais ou français.
    • Profils : Les normes internationales que vous pouvez sélectionner sont les suiivantes : IEEE LOM, SCORM, CANCORE et Normetic.
    • Aide : IEEE document.
  • La section de métadonnées permet de référencer les objets d'apprentissage selon les métadonnées du LOM.

2. Gestion des LOMs

Vous pouvez avec LomPad consulter, créer, modifier, sauver des objets d'apprentissage selon les métadonnées du LOM.

2.1 Ajouter un LOM

La commande Nouveau du menu Fichier permet d'ajouter un nouveau LOM.

2.2 Compléter les métadonnées

Le LomPad permet de référencer les ressources dans neuf grandes catégories à l'aide de métadonnées selon les normes internationales (IEEE LOM, SCORM, CANCORE et Normetic).

  1. La catégorie Général regroupe les informations générales qui décrivent l'objet d'apprentissage dans son l'ensemble.
  2. La catégorie Cycle de vie regroupe les dispositifs liés à l'histoire et à l'état actuel de cet objet d'apprentissage et ceux qui ont affecté cet objet d'étude pendant son évolution.
  3. La catégorie de Méta-Metadata regroupe des informations sur les instances des métadatas elle-même (plutôt que l'objet d'apprentissage).
  4. La catégorie Technique regroupe les impératifs techniques et les caractéristiques techniques de l'objet d'apprentissage.
  5. La catégorie Éducationnel regroupe les caractéristiques éducatives et pédagogiques de l'objet d'apprentissage.
  6. La catégorie Droits regroupe les droites de propriété intellectuelle et les conditions de l'utilisation de l'objet d'apprentissage.
  7. La catégorie de Relation regroupe les dispositifs qui permettent de définir le rapport entre cet objet d'apprentissage et les autres les objets d'apprentissage.
  8. La catégorie Annotation fournit des commentaires sur l'utilisation éducative de l'objet d'apprentissage et fournit des informations sur la date de création et l'auteur des commentaires..
  9. La catégorie Classification décrit cet objet d'apprentissage par rapport à un système de classification particulier.
  • Il est possible d'ajouter à chaque LOM plusieurs cardinalités aux catégories suivantes : Éducationel, relation, Annotation et Classification.
  • Pour référencer les objets d'apprentissage selon les métadonnées du LOM, vous entrez les informations dans les bons éléments.
  • Cliquez sur le bouton Sauver.

2.2.1 Ajouter un élément dans une catégorie

  • Sélectionnez une catégorie.
  • Cliquez sur le bouton ajouter (+) à droite de l'élément.

2.2.2 Supprimer un élément dans une catégorie

  • Sélectionnez une catégorie
  • Cliquez sur le bouton supprimer (-) à droite de l'élément.

2.2.3 Ajouter une nouvelle catégorie à un LOM

Les catégories suivantes possèdent une cardinalité multiple : Éducationnel, Relation, Annotation, et Classification.

  • Sélectionnez une des catégories suivantes : Éducationnel, Relation, Annotation, et Classification.
  • Cliquez sur le bouton ajouter (+) à gauche de la fenêtre de la catégorie.

2.2.4 Supprimer une catégorie d'un LOM

  • Sélectionnez une des catégories suivantes : Educationnel, Relation, Annotation, et Classification.
  • Cliquez sur le bouton supprimer (-) à gauche de la fenêtre de la catégorie.

2.3 Ouvrir le contenu d'un LOM

La commande Ouvrir du menu Fichier ouvre une boîte de dialogue qui vous permet de localiser et d'ouvrir un LOM à partir de son nom de fichier XML.

2.4 Sauver le contenu d'un LOM

La commande Sauver ou Enregistrer Sous du menu Fichier ouvre une boîte de dialogue Windows® qui vous permet de localiser et de sauver le contenu d'un LOM à partir de son nom de fichier avec un format XML.

2.5 Voir les métadonnées du LOM en XML

  • Sélectionnez un object d'apprentissage.
  • Cliquez sur le bouton XML.

2.6 Quitter LomPad

  • Cliquez sur le bouton Quitter.
